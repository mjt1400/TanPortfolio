“Smart Academic Guidance Engine": A Data Analytics Tool for the Academic Staff of Mapúa Malayan Colleges Mindanao (MMCM)

Leveraging the power of the MERN stack, this tool harnesses data from the Blackboard Learning Management System to provide insightful analytics and guidance. Explore how this innovative solution empowers academic staff with actionable data-driven insights for optimizing student performance and enhancing the learning experience at MMCM.

Web Appointment System for A Therapy Center

Developed a comprehensive Web Appointment System for a Therapy Center, utilizing a stack comprising HTML, CSS, JavaScript, PHP, and SQL. This system streamlines appointment scheduling, client management, and administrative tasks, ensuring a seamless experience for both therapists and clients.

King's Cup Game

The "King's Cup Game" is a web-based drinking game developed using HTML, CSS, and JavaScript, featuring a deck of cards sourced from an API specifically designed for card games. Players can enjoy the classic King's Cup experience virtually, with interactive card draws, game rules, and a dynamic user interface.

Heart Disease Classification App

In this project, machine learning was utilized to make predictions about a person's risk of acquiring heart disease based on a variety of characteristics. It also utilizes Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, etc, in order to organize & visualize the data, and to produce a model.

GPT on the Go: ChatGPT Voice Assistant Device for Students

This is a project that combines the power of artificial intelligence with the flexibility of Raspberry Pi hardware. By integrating the ChatGPT model into a Raspberry Pi device, this project enables conversational interactions and responses based on natural language processing. The Chatbot can perform a wide range of tasks, from answering queries to controlling connected devices, offering a seamless and intuitive user experience.

Twitter Clone Mobile App

This is a project developed using Dart and Flutter frameworks in Android Studio. It replicates the core functionalities of Twitter, allowing users to post tweets, follow other users, and engage with content. With a sleek and responsive design, this app offers a modern social media experience tailored for mobile devices.
